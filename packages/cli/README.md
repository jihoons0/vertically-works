# verticallyworks

Copy-in components for vertical writing interfaces — design tokens and vertical-first React components from [vertically.works](https://vertically.works).

```bash
# install the design tokens (components/vw/tokens.css)
npx verticallyworks init

# copy a component into your project
npx verticallyworks add vertical-button

# see everything available
npx verticallyworks list
```

Components are source files you own: real `writing-mode: vertical-rl`, tokens-only styling, keyboard and screen-reader support, reduced-motion aware. No runtime dependency.

Full documentation, live demos, and the design reasoning behind every component: **https://vertically.works**
